# UESTC_Compiler_Experiment_Implemented_in_Python

电子科技大学 编译原理实验 廖子牧

使用Python实现完成类Pascal语言的词法分析和语法分析


如果需要运行程序
```angular2html
python main.py
```

如果需要分析其他路径下的源码
```angular2html
python main.py --file_path your_source_code_path
```

如果只需要完成词法分析
```angular2html
python main.py --lexical_only True
```
